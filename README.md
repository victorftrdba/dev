## Rodando a Aplicação

Para rodar a aplicação faça um git clone com o link do meu repositório através do seu terminal e depois de clonado digite os seguintes comandos:

- composer install
- npm install
- php artisan serve (roda a aplicação)

Lembre-se de Ativar todas as Extensões necessárias no php.ini!
